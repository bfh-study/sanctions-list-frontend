Add RestService to your Webshop!

On this page we explain how do you can use our restservice in your Webshop.

How to add to your website


  1. Add the follwing URL to your code: http://localhost:8080/api/allowed/firstName/lastName
  2. the PathParam firstName and lastName must set by you
  3. if the person is in our DB, it will return TRUE (as String)
  4. if the person is not listed in your DB, it will return FALSE (also as String)
